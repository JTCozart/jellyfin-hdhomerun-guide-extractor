Your HDHomeRun's guide, already in Jellyfin

A Jellyfin plugin that generates an auto-refreshing XMLTV guide from your HDHomeRun's own EPG, with channel ids pre-mapped to your tuner lineup. No manual mapping, no external scripts, no cron jobs.

Jellyfin's XMLTV guide provider

Set it up once, forget it

Uses Jellyfin's own built-in XMLTV guide provider — this plugin just keeps the file it reads current.

📡

Auto channel mapping

Matches each lineup entry's guide number to the feed by display name, so channels line up with your tuner with zero manual mapping.

🔁

Scheduled refresh

Regenerates the guide on a configurable interval (default 12 hours) — the file just stays current.

🗂

Multiple devices

Configure more than one HDHomeRun and their guides are merged into a single file, deduplicated by channel.

🧩

No extra infrastructure

No external scripts, no cron jobs, no separate container — it runs inside Jellyfin as a background service.

📊

Live status

The plugin page shows channel/programme counts, last success, and any errors, plus a one-click "Refresh now".

🔒

Admin-only API

Status and refresh are exposed under /HdhrEpg, gated behind Jellyfin's own elevated-admin policy.

Install in two minutes

Add the repository once and install (plus future updates) straight from Jellyfin's catalog.

Open repositories. In Jellyfin go to Dashboard → Plugins → Repositories and click ＋.
Add this repository URL.
https://raw.githubusercontent.com/JTCozart/jellyfin-hdhomerun-guide-extractor/master/manifest.json
Install the plugin. Open Catalog, find HDHomeRun EPG under Live TV, and click Install.
Restart Jellyfin, open HDHomeRun EPG from the Dashboard menu, set your device host(s), save, and click Refresh now.
Point the guide provider at the file. Go to Dashboard → Live TV → Guide Data Providers → Add, choose XMLTV, and paste in the file path shown on the plugin's page.
